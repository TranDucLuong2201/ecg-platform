Getting Started

This platform provides tools for ECG signal analysis, including FIR and IIR filtering, spectral analysis, and PDF report generation.

API Endpoints

  • /api/analyze: Analyze ECG signals with specified parameters.
  • /generate_pdf: Generate a PDF report of the analysis.
  • /download/<image_type>: Download individual plot images (frequency_response, time_domain, frequency_domain).

Filter Types

Supported filters include:

  • FIR: Hamming, Hann, Blackman, Kaiser windows
  • IIR: Butterworth, Chebyshev I, Chebyshev II, Elliptic

Sample CSV Format

Ensure your CSV includes columns like 'MLII' or other numeric data for ECG signals.
